#Cryptid Cabin

#Installation Instructions:

* `npm i`
* create a  `.env` file in the root fo the project
* add the following to the `.env` file:

```
DATABASE_URL=
    mongodb+srv://admin:
        <PASSWORD>@cluster0.m0exe.mongodb.net/
        <DATABASE_NAME>?retryWrites=true&w=majority

SECRET=<STRING>
```
